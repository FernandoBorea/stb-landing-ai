# NanoBanana Pro Prompts for STB Landing Page Mockups

## Prompt 1: Home Page Desktop View

Create a professional desktop landing page mockup for "Smoothies The Best" (Smoothies The Best), a B2B smoothie company targeting restaurants and hotels.

### Design Requirements:
- **Overall aesthetic**: Clean, modern, product-focused layout inspired by Hartzler Dairy (ample white space, large typography, strong product imagery)
- **Color palette**:
  - Primary Blue: `#1F6EAD`
  - Primary Orange: `#E9891F`
  - Accent Orange: `#F15A24`
  - White: `#FFFFFF`
  - Light Gray: `#F5F5F5` (for carousel/headline section)
- **Typography**: Use Circular Std font family (or similar modern sans-serif) - bold for headlines, regular for body text
- **Language**: All content in Spanish

### Layout Structure (top to bottom):

1. **Header/Navigation Bar**:
   - Light blue background (20% opacity of primary blue)
   - Left: Hamburger menu icon (3 horizontal lines) in blue
   - Center: "SMOOTHIES THE BEST" logo (blue)
   - Right: Empty space (for symmetry)

2. **Hero Section**:
   - Solid blue background (`#1F6EAD`)
   - Three large gallon bottles displayed horizontally, centered
   - Each bottle has white text label ABOVE it (not beside):
     - Left: "Sandía" (watermelon - pink/red liquid)
     - Center: "Piña Colada" (pale yellow/cream liquid)
     - Right: "Café" (brown/coffee liquid)
   - Generous vertical padding

3. **Carousel & Headline Section**:
   - Light gray background (`#F5F5F5`)
   - Top: Multiple "SMOOTHIES THE BEST" logos repeated horizontally (faded/gray, varying opacity 30-40%)
   - Bottom: Large centered headline "NATURAL & FRESH" in bold blue (`#1F6EAD`), very large font size

4. **Video/CTA Section**:
   - Full-width section with video background (factory/production facility)
   - Strong blue overlay (`#1F6EAD` at 70% opacity) over video
   - Centered white text:
     - "Bienvenidos a" (smaller)
     - "Smoothies The Best" (large, bold)
   - White button with blue text: "Contáctanos" (rounded corners, prominent)

5. **Footer**:
   - White background
   - Horizontal layout: Logo (left) | 4 buttons (center) | WhatsApp icon (right)
   - Buttons: Blue background, white text, rounded: "Productos", "¿Por qué Elegirnos?", "Clientes", "Contacto"
   - WhatsApp icon: Green circular icon on the right

### Assets to Provide:
- `blue_logo.png` - STB logo in blue
- `watermelon.png` - Gallon bottle with watermelon smoothie
- `pina_colada.png` - Gallon bottle with piña colada smoothie
- `coffee.png` - Gallon bottle with coffee smoothie
- `stb_factory.mp4` or screenshot - Video/background for CTA section
- Brand color palette reference (hex codes above)

### Output Requirements:
- Desktop viewport: 1920x1080 or similar wide format
- High fidelity mockup showing exact layout, spacing, and typography
- Export as PNG or PDF

## Prompt 3: Menu Overlay Desktop View

Create a full-screen menu overlay mockup for desktop view, inspired by b-egg.farm menu design but adapted for Smoothies The Best branding.

### Design Requirements:
- **Style**: Full-screen overlay, modern and bold
- **Color palette**: 
  - Background: Solid blue (`#1F6EAD`) at 95% opacity
  - Text: White
  - Accent: Orange (`#E9891F`) on hover
- **Typography**: Circular Std, EXTRA LARGE for menu items (oversized, bold)
- **Language**: Spanish
- **Viewport**: Desktop (1920x1080)

### Layout Structure:

1. **Top Section**:
   - Top-left: White "SMOOTHIES THE BEST" logo
   - Top-right: Black "X" close button (large, prominent)

2. **Center Section**:
   - Vertically centered menu items, left-aligned
   - Four oversized menu links in white, bold, very large font:
     - "Productos"
     - "¿Por qué Elegirnos?"
     - "Clientes"
     - "Contacto"
   - Generous spacing between items (60-80px)
   - Hover state: Orange color (`#E9891F`)

3. **Bottom Section**:
   - Decorative product image on right side (bottom-right corner)
   - Image shows five gallon bottles with fresh ingredients (coconut, passion fruit, watermelon, pineapple, chocolate)
   - Image at 30% opacity, partially covering bottom-right area
   - Image should not interfere with menu readability

### Assets to Provide:
- `white_logo.png` - STB logo in white
- `five_gallons.png` - Decorative product image with 5 bottles and ingredients
- Brand color palette

### Output Requirements:
- Full-screen overlay mockup (1920x1080)
- Show menu in open state
- Export as PNG
